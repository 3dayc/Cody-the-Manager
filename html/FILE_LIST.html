작업현황판 작성 규칙
  • 상태표기 : '△'(작업중대기) , '○'(부분완료) , '●'(완료) 중 알맞은 것으로 표기. '●'(완료) 표기부터 개발작업 가능.
  • 날짜의 형식은 yyyy-mm-dd 으로 하며, HTML 파일 내용이 수정되었을 경우, 최초작성일('●'(완료)일 기준)은 남겨두고 옆에 마지막 수정일자를 표기함. (예) 2013-04-26 2013-05-16)
  • 'description'란은 화면에 대한 추가적인 설명, 'note'란은 작업자들 간에 서로 알 수 있는 메모(notice)를 기재함.
